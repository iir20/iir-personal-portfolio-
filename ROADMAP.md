## 🔮 PHASE 2: AI REINFORCEMENTS (Planned for Q3 2026)

### Smart AI Assistant Integration
* Integrate the **Gemini 2.5 Flash SDK** server-side to power a real-time conversational helper on the site.
* Train the agent with specialized portfolio context (project details, resume, skills matrix, and security policies) to answer questions from recruiters.

### Automated CV Translation and Summary
* Build an AI-assisted resume summarizer that compiles custom, industry-specific summaries based on the visitor’s background (e.g., tailoring CV insights for a Cybersecurity lead vs. a Frontend developer).

---

## 🎨 PHASE 3: ADVANCED INTERFACES & ACCESSIBILITY (Target Q4 2026)

### Full WCAG 2.2 AA Accessibility Compliance
* Audit color contrast indices across all neon terminal layouts to guarantee a minimum contrast ratio of `4.5:1` for normal text and `3:1` for headers.
* Establish comprehensive keyboard navigation boundaries (`Tab` routing and `Escape` handlers) for all dialog panels, drawers, and command lines.

### Immersive Audio Environment
* Expand the custom web audio synthesizer into a full-scale cyberpunk ambient background music engine.
* Add user-toggleable white/pink noise loops, mechanical keyboard click sounds for keyboard interactions, and retro diagnostic terminal chirps.

---

## ⚡ PHASE 4: PERFORMANCE & DATA ENGINE OPTIMIZATIONS (Target Q1 2027)

### Native Edge Cache Buffering
* Leverage Cloudflare workers or secondary service workers to cache portfolio states locally on edge servers, targeting worldwide load times under `200ms`.
* Compile compressed, vectorized SVG equivalents of all heavy icon and graphical assets to shrink initial bundles.

### Secure Relational Database Migration (Cloud SQL / Drizzle)
* Shift from Firestore's NoSQL structures to a relational database scheme via Cloud SQL PostgreSQL and Drizzle ORM to compile complex statistics (e.g., aggregate page view rates, device trackers, and geographic login maps).
