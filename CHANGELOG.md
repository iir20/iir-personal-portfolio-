# 📋 CHANGELOG — RATUL'S CYBERSPACE DIGITAL PORTFOLIO OS

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-06-27
### Added
- **Final Release Preparation**: Automated CI/CD pipeline inside `.github/workflows/deploy.yml` with cache restoring, TypeScript type validation, ESLint assertions, sitemap generation, and native GitHub Pages deployment records.
- **Repository Standardizations**: Integrated standard open-source metadata descriptors including `.editorconfig`, `.gitattributes`, `.prettierrc`, `.prettierignore`, `CODEOWNERS`, `SUPPORT.md`, `CONTRIBUTING.md`, `ROADMAP.md`, `CODE_OF_CONDUCT.md`, and `robots.txt`.
- **Progressive Web App (PWA)**: Implemented PWA `manifest.webmanifest`, high-contrast vector `favicon.svg`, standard tile setups inside `browserconfig.xml`, and baseline binary icon fallbacks.
- **Deep Routing Fallbacks**: Built a custom styled `404.html` fallback with a retro CRT monitor terminal look and auto-navigation to manage deep linking on static host CDNs.
- **IPS Security Center logs**: Enhanced the local telemetry list to preserve incident logs into `localStorage` across page updates.

---

## [0.9.0] - 2026-06-20
### Added
- **Dynamic Backup Import & Export Panel**: Designed an administrative control deck to package entire CMS collections, CV database pointers, and security configurations into a single JSON file.
- **Client-Side Incident Monitoring Listeners**: Attached event hooks for Content-Security-Policy (CSP) violations, uncaught runtime script errors, unhandled promise rejections, and DevTools inspection triggers to log and display alerts inside the SOC telemetry dashboard.
- **Manual Sync Rate Limiting**: Added click trackers to prevent excessive manual sync triggers in under 10 seconds.
- **Secure MIME Headers**: Formatted standard strict CSP meta headers in `/index.html` to mitigate cross-site scripting (XSS) and injection attempts.

---

## [0.8.0] - 2026-06-12
### Added
- **Secure CV Attachment & Versioning**: Developed a complete PDF CV Manager. Added double-layered size validation limits (5MB) and version tracking up to 5 historical rollbacks.
- **Admin Authentication Hardening**: Implemented progressive throttling, account lockouts for 30 seconds after 3 failed handshakes, and login threat logging inside the SOC console.

## [0.7.0] - 2026-06-05

- **GitHub API Integration**: Formatted automated sync connectors to fetch user profiles, followers, star counts, and repository metrics directly from GitHub REST APIs.
- **Upcoming Repository Filter**: Designed metadata categorizers to separate completed projects from WIP (Work in Progress) repositories automatically in the portfolio feed.
